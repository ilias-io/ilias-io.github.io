What is ilias?
Quick Start
API Reference

Lightweight

Minimal core dependencies, relying only on the standard library and system headers.

Modern

Completion-based model, rich Tokio-like APIs, and built-in support for asynchronous cancellation and structured concurrency.

Simple

One-scheduler-per-thread design and easy-to-use APIs significantly reduce cognitive load.

Interoperable

Supports interoperability with Qt, making it very easy to integrate into existing code and other event frameworks (such as libuv, Boost.Asio).
